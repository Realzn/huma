# Contribuer à HUMA

HUMA appartient à tout le monde. Toute contribution est bienvenue.

## Pour contribuer du code

```bash
git clone https://github.com/TON_USERNAME/huma
cd huma
npm install
cp apps/web/.env.example apps/web/.env.local
# Remplir les clés Supabase
npm run dev
```

## Structure des commits

```
feat: nouvelle fonctionnalité
fix: correction de bug
docs: documentation
style: mise en forme
refactor: refactoring
```
